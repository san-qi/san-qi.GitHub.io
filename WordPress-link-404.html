WordPress页面链接404问题

  • Wed 27 May 2020
  • wordpress
  • #WordPress

说明

可能我们出于一些原因会更改WordPress的一项设置，该项设置的更改大概率会导致原本能够访问的页面链接突然不能访问

该项设置位于WordPress后台中 设置-固定链接-固定链接设置。

解决

最初始的设置选项是可以正常访问链接的，但是当我们对这项设置的选项值更改后，要想继续正常访问，我们需要对nginx或者是apache的配置文件进行更改 我的服务器使用的是nginx apache还没使用的经验， 我们在nginx配置文件对应域名（你用来提供WordPress服务的子域名）的server块加上这样一项配置，这可能需要你对nginx的配置有所熟悉。

location /
{
     try_files $uri $uri/ /index.php?$args;
}

rewrite /wp-admin$ $scheme://$host$uri/ permanent;

配置完成后apt/yum update一下，应该能让WordPress链接正常指向了
